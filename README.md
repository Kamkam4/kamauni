# უნივერსიტეტის საიტი

ეს არის თანამედროვე დიზაინის უნივერსიტეტის ვებ აპლიკაცია, რომელიც საშუალებას აძლევს სტუდენტებს რეგისტრაცია გაიარონ და მათი შეფასებები დაინახონ.

## ფუნქციები

- **რეგისტრაცია**: სტუდენტები შეუძლიათ რეგისტრაცია სახელი, გვარი, დაბადების წელი, ფოტო და ფაკულტეტის არჩევით.
- **შესვლა**: ელ-ფოსტით (სახელი.გვარი@kamauni.ge) და პაროლით (kamauni2025).
- **Dashboard**: სტუდენტების პროფილი და შეფასებების ნახვა ფაკულტეტის საგნებზე.

## დაწყება

პირველ რიგში, დააინსტალირეთ დამოკიდებულებები:

```bash
npm install
```

შემდეგ, გაუშვით მონაცემთა ბაზის სიდი:

```bash
npx tsx src/lib/seed.ts
```

და გაუშვით დეველოპმენტ სერვერი:

```bash
npm run dev
```

გახსენით [http://localhost:3000](http://localhost:3000) ბრაუზერში.
