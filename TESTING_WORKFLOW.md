# рЃАрЃбрЃБрЃЊрЃћрЃюрЃбрЃБрЃарЃў рЃЏрЃљрЃарЃЌрЃЋрЃўрЃА рЃАрЃўрЃАрЃбрЃћрЃЏрЃўрЃА рЃбрЃћрЃАрЃбрЃўрЃарЃћрЃЉрЃўрЃА рЃњрЃћрЃњрЃЏрЃљ

## ­ЪЊІ рЃбрЃћрЃАрЃбрЃўрЃарЃћрЃЉрЃўрЃА рЃЏрЃўрЃЏрЃЮрЃ«рЃўрЃџрЃЋрЃљ

рЃћрЃА рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃў рЃерЃћрЃўрЃфрЃљрЃЋрЃА рЃАрЃарЃБрЃџ рЃбрЃћрЃАрЃбрЃўрЃарЃћрЃЉрЃўрЃА рЃњрЃћрЃњрЃЏрЃљрЃА рЃАрЃбрЃБрЃЊрЃћрЃюрЃбрЃБрЃарЃў рЃЏрЃљрЃарЃЌрЃЋрЃўрЃА рЃАрЃўрЃАрЃбрЃћрЃЏрЃўрЃАрЃЌрЃЋрЃўрЃА, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃЏрЃЮрЃўрЃфрЃљрЃЋрЃА рЃарЃћрЃњрЃўрЃАрЃбрЃарЃљрЃфрЃўрЃљрЃА, рЃљрЃЋрЃбрЃЮрЃарЃўрЃќрЃљрЃфрЃўрЃљрЃА, рЃЊрЃћрЃерЃЉрЃЮрЃарЃЊрЃА, рЃюрЃўрЃерЃюрЃћрЃЉрЃўрЃА рЃЕрЃЋрЃћрЃюрЃћрЃЉрЃљрЃА, рЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃА рЃерЃћрЃерЃўрЃюрЃћрЃЉрЃљрЃА рЃЊрЃљ рЃЊрЃўрЃърЃџрЃЮрЃЏрЃўрЃА рЃњрЃћрЃюрЃћрЃарЃљрЃфрЃўрЃљрЃА.

## ­ЪћД рЃгрЃўрЃюрЃљрЃърЃўрЃарЃЮрЃЉрЃћрЃЉрЃў

рЃАрЃљрЃюрЃљрЃЏ рЃЊрЃљрЃўрЃгрЃДрЃћрЃЉрЃЌ рЃбрЃћрЃАрЃбрЃўрЃарЃћрЃЉрЃљрЃА, рЃЊрЃљрЃарЃгрЃЏрЃБрЃюрЃЊрЃўрЃЌ рЃарЃЮрЃЏ:

1. **Vercel Blob Storage** рЃЏрЃЮрЃарЃњрЃћрЃЉрЃБрЃџрЃўрЃљ рЃЊрЃљ `BLOB_READ_WRITE_TOKEN` рЃњрЃљрЃарЃћрЃЏрЃЮрЃА рЃфрЃЋрЃџрЃљрЃЊрЃў рЃЊрЃљрЃДрЃћрЃюрЃћрЃЉрЃБрЃџрЃўрЃљ
2. **PostgreSQL рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЉрЃљрЃќрЃљ** рЃЊрЃљрЃЎрЃљрЃЋрЃерЃўрЃарЃћрЃЉрЃБрЃџрЃўрЃљ рЃЊрЃљ рЃЊрЃљрЃАрЃўрЃЊрЃБрЃџрЃўрЃљ
3. **рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃљ рЃЊрЃљрЃърЃџрЃЮрЃўрЃюрЃбрЃћрЃЉрЃБрЃџрЃўрЃљ** Vercel-рЃќрЃћ
4. **API рЃћрЃюрЃЊрЃърЃЮрЃўрЃюрЃбрЃћрЃЉрЃў** рЃЏрЃБрЃерЃљрЃЮрЃЉрЃћрЃю (`/api/health`, `/api/faculties`)

## ­ЪДф рЃбрЃћрЃАрЃбрЃўрЃарЃћрЃЉрЃўрЃА рЃАрЃфрЃћрЃюрЃљрЃарЃћрЃЉрЃў

### 1. рЃарЃћрЃњрЃўрЃАрЃбрЃарЃљрЃфрЃўрЃўрЃА рЃбрЃћрЃАрЃбрЃўрЃарЃћрЃЉрЃљ

#### 1.1 рЃгрЃљрЃарЃЏрЃљрЃбрЃћрЃЉрЃБрЃџрЃў рЃарЃћрЃњрЃўрЃАрЃбрЃарЃљрЃфрЃўрЃљ
- **рЃЏрЃЮрЃЦрЃЏрЃћрЃЊрЃћрЃЉрЃћрЃЉрЃў:**
  1. рЃњрЃљрЃЊрЃљрЃЊрЃўрЃЌ `/register` рЃњрЃЋрЃћрЃарЃЊрЃќрЃћ
  2. рЃерЃћрЃљрЃЋрЃАрЃћрЃЌ рЃцрЃЮрЃарЃЏрЃљ:
     - рЃАрЃљрЃ«рЃћрЃџрЃў: "рЃњрЃўрЃЮрЃарЃњрЃў"
     - рЃњрЃЋрЃљрЃарЃў: "рЃЏрЃљрЃўрЃАрЃБрЃарЃљрЃФрЃћ"
     - рЃЊрЃљрЃЉрЃљрЃЊрЃћрЃЉрЃўрЃА рЃЌрЃљрЃарЃўрЃдрЃў: "2000-01-01"
     - рЃцрЃљрЃЎрЃБрЃџрЃбрЃћрЃбрЃў: рЃљрЃўрЃарЃЕрЃўрЃћрЃЌ рЃюрЃћрЃЉрЃўрЃАрЃЏрЃўрЃћрЃарЃў
     - рЃцрЃЮрЃбрЃЮ: рЃљрЃбрЃЋрЃўрЃарЃЌрЃћрЃЌ рЃАрЃБрЃарЃљрЃЌрЃў (JPG/PNG, < 5MB)
  3. рЃЊрЃљрЃљрЃГрЃўрЃарЃћрЃЌ "рЃарЃћрЃњрЃўрЃАрЃбрЃарЃљрЃфрЃўрЃљ" рЃдрЃўрЃџрЃљрЃЎрЃА

- **рЃЏрЃЮрЃАрЃљрЃџрЃЮрЃЊрЃюрЃћрЃџрЃў рЃерЃћрЃЊрЃћрЃњрЃў:**
  - рЃгрЃљрЃарЃЏрЃљрЃбрЃћрЃЉрЃўрЃА рЃерЃћрЃбрЃДрЃЮрЃЉрЃўрЃюрЃћрЃЉрЃљ
  - рЃњрЃљрЃЊрЃљрЃЏрЃўрЃАрЃљрЃЏрЃљрЃарЃЌрЃћрЃЉрЃљ `/login` рЃњрЃЋрЃћрЃарЃЊрЃќрЃћ
  - рЃћрЃџ-рЃцрЃЮрЃАрЃбрЃљ рЃЊрЃљ рЃърЃљрЃарЃЮрЃџрЃў рЃњрЃћрЃюрЃћрЃарЃўрЃарЃЊрЃћрЃЉрЃљ рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ

#### 1.2 рЃЋрЃљрЃџрЃўрЃЊрЃљрЃфрЃўрЃўрЃА рЃбрЃћрЃАрЃбрЃўрЃарЃћрЃЉрЃљ
- **рЃфрЃљрЃарЃўрЃћрЃџрЃў рЃЋрЃћрЃџрЃћрЃЉрЃў:** рЃерЃћрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЌ рЃарЃЮрЃЏ рЃДрЃЋрЃћрЃџрЃљ рЃЋрЃћрЃџрЃў рЃАрЃљрЃЋрЃљрЃџрЃЊрЃћрЃЉрЃБрЃџрЃЮрЃљ
- **рЃљрЃарЃљрЃАрЃгрЃЮрЃарЃў рЃцрЃЮрЃарЃЏрЃљрЃбрЃў:** рЃЊрЃљрЃЉрЃљрЃЊрЃћрЃЉрЃўрЃА рЃЌрЃљрЃарЃўрЃдрЃў, рЃцрЃЮрЃбрЃЮрЃА рЃќрЃЮрЃЏрЃљ
- **рЃЊрЃБрЃЉрЃџрЃўрЃарЃћрЃЉрЃБрЃџрЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃў:** рЃўрЃњрЃўрЃЋрЃћ рЃАрЃљрЃ«рЃћрЃџрЃў/рЃњрЃЋрЃљрЃарЃў

### 2. рЃљрЃЋрЃбрЃЮрЃарЃўрЃќрЃљрЃфрЃўрЃўрЃА рЃбрЃћрЃАрЃбрЃўрЃарЃћрЃЉрЃљ

#### 2.1 рЃгрЃљрЃарЃЏрЃљрЃбрЃћрЃЉрЃБрЃџрЃў рЃерЃћрЃАрЃЋрЃџрЃљ
- **рЃЏрЃЮрЃЦрЃЏрЃћрЃЊрЃћрЃЉрЃћрЃЉрЃў:**
  1. рЃњрЃљрЃЊрЃљрЃЊрЃўрЃЌ `/login` рЃњрЃЋрЃћрЃарЃЊрЃќрЃћ
  2. рЃерЃћрЃўрЃДрЃЋрЃљрЃюрЃћрЃЌ рЃарЃћрЃњрЃўрЃАрЃбрЃарЃљрЃфрЃўрЃўрЃА рЃЊрЃарЃЮрЃА рЃЏрЃўрЃдрЃћрЃЉрЃБрЃџрЃў рЃћрЃџ-рЃцрЃЮрЃАрЃбрЃљ рЃЊрЃљ рЃърЃљрЃарЃЮрЃџрЃў
  3. рЃЊрЃљрЃљрЃГрЃўрЃарЃћрЃЌ "рЃерЃћрЃАрЃЋрЃџрЃљ" рЃдрЃўрЃџрЃљрЃЎрЃА

- **рЃЏрЃЮрЃАрЃљрЃџрЃЮрЃЊрЃюрЃћрЃџрЃў рЃерЃћрЃЊрЃћрЃњрЃў:**
  - рЃњрЃљрЃЊрЃљрЃЏрЃўрЃАрЃљрЃЏрЃљрЃарЃЌрЃћрЃЉрЃљ `/dashboard` рЃњрЃЋрЃћрЃарЃЊрЃќрЃћ
  - рЃАрЃбрЃБрЃЊрЃћрЃюрЃбрЃўрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃЕрЃЋрЃћрЃюрЃћрЃЉрЃљ

#### 2.2 рЃљрЃарЃљрЃАрЃгрЃЮрЃарЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃў
- **рЃљрЃарЃљрЃАрЃгрЃЮрЃарЃў рЃћрЃџ-рЃцрЃЮрЃАрЃбрЃљ/рЃърЃљрЃарЃЮрЃџрЃў:** рЃерЃћрЃбрЃДрЃЮрЃЉрЃўрЃюрЃћрЃЉрЃљ "рЃљрЃарЃљрЃАрЃгрЃЮрЃарЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃў"
- **рЃфрЃљрЃарЃўрЃћрЃџрЃў рЃЋрЃћрЃџрЃћрЃЉрЃў:** рЃЋрЃљрЃџрЃўрЃЊрЃљрЃфрЃўрЃўрЃА рЃерЃћрЃбрЃДрЃЮрЃЉрЃўрЃюрЃћрЃЉрЃћрЃЉрЃў

### 3. рЃЊрЃћрЃерЃЉрЃЮрЃарЃЊрЃўрЃА рЃбрЃћрЃАрЃбрЃўрЃарЃћрЃЉрЃљ

#### 3.1 рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃЕрЃЋрЃћрЃюрЃћрЃЉрЃљ
- **рЃЏрЃЮрЃАрЃљрЃџрЃЮрЃЊрЃюрЃћрЃџрЃў рЃерЃћрЃЊрЃћрЃњрЃў:**
  - рЃАрЃбрЃБрЃЊрЃћрЃюрЃбрЃўрЃА рЃАрЃљрЃ«рЃћрЃџрЃў, рЃњрЃЋрЃљрЃарЃў, рЃцрЃљрЃЎрЃБрЃџрЃбрЃћрЃбрЃў
  - рЃАрЃљрЃерЃБрЃљрЃџрЃЮ рЃЦрЃБрЃџрЃљ (рЃЌрЃљрЃЋрЃЊрЃљрЃърЃўрЃарЃЋрЃћрЃџрЃљрЃЊ 0.00)
  - рЃюрЃўрЃерЃюрЃћрЃЉрЃўрЃА рЃфрЃ«рЃарЃўрЃџрЃў (рЃЌрЃљрЃЋрЃЊрЃљрЃърЃўрЃарЃЋрЃћрЃџрЃљрЃЊ рЃфрЃљрЃарЃўрЃћрЃџрЃў)
  - рЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃА рЃерЃћрЃерЃўрЃюрЃћрЃЉрЃўрЃА рЃдрЃўрЃџрЃљрЃЎрЃў
  - рЃЊрЃўрЃърЃџрЃЮрЃЏрЃўрЃА рЃњрЃћрЃюрЃћрЃарЃљрЃфрЃўрЃўрЃА рЃдрЃўрЃџрЃљрЃЎрЃў

#### 3.2 рЃюрЃљрЃЋрЃўрЃњрЃљрЃфрЃўрЃљ
- **рЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃА рЃерЃћрЃерЃўрЃюрЃћрЃЉрЃљ:** рЃњрЃљрЃЊрЃљрЃАрЃЋрЃџрЃљ `/scare-rector` рЃњрЃЋрЃћрЃарЃЊрЃќрЃћ
- **рЃЊрЃўрЃърЃџрЃЮрЃЏрЃў:** рЃњрЃљрЃЊрЃљрЃАрЃЋрЃџрЃљ `/diploma` рЃњрЃЋрЃћрЃарЃЊрЃќрЃћ

### 4. рЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃА рЃерЃћрЃерЃўрЃюрЃћрЃЉрЃўрЃА рЃбрЃћрЃАрЃбрЃўрЃарЃћрЃЉрЃљ

#### 4.1 рЃЌрЃљрЃЏрЃљрЃерЃўрЃА рЃЏрЃўрЃЏрЃЊрЃўрЃюрЃљрЃарЃћрЃЮрЃЉрЃљ
- **рЃЏрЃЮрЃЦрЃЏрЃћрЃЊрЃћрЃЉрЃћрЃЉрЃў:**
  1. рЃЊрЃљрЃљрЃГрЃўрЃарЃћрЃЌ "рЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃА рЃерЃћрЃерЃўрЃюрЃћрЃЉрЃљ" рЃдрЃўрЃџрЃљрЃЎрЃА рЃЊрЃћрЃерЃЉрЃЮрЃарЃЊрЃќрЃћ
  2. рЃЊрЃљрЃљрЃГрЃўрЃарЃћрЃЌ "рЃЊрЃљрЃўрЃгрЃДрЃћ рЃерЃћрЃерЃўрЃюрЃћрЃЉрЃљ" рЃдрЃўрЃџрЃљрЃЎрЃА
  3. рЃЏрЃўрЃ░рЃДрЃћрЃЋрЃўрЃЌ рЃўрЃюрЃАрЃбрЃарЃБрЃЦрЃфрЃўрЃћрЃЉрЃА (рЃЎрЃџрЃљрЃЋрЃўрЃљрЃбрЃБрЃарЃўрЃА рЃдрЃўрЃџрЃљрЃЎрЃћрЃЉрЃў)
  4. рЃЊрЃљрЃљрЃАрЃарЃБрЃџрЃћрЃЌ рЃЌрЃљрЃЏрЃљрЃерЃў

- **рЃЏрЃЮрЃАрЃљрЃџрЃЮрЃЊрЃюрЃћрЃџрЃў рЃерЃћрЃЊрЃћрЃњрЃў:**
  - рЃЦрЃБрЃџрЃўрЃА рЃЏрЃЮрЃЏрЃљрЃбрЃћрЃЉрЃљ (1-5 рЃЦрЃБрЃџрЃљ)
  - рЃњрЃљрЃЊрЃљрЃЏрЃўрЃАрЃљрЃЏрЃљрЃарЃЌрЃћрЃЉрЃљ рЃЊрЃћрЃерЃЉрЃЮрЃарЃЊрЃќрЃћ
  - рЃњрЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃБрЃџрЃў рЃАрЃљрЃерЃБрЃљрЃџрЃЮ рЃЦрЃБрЃџрЃљ

#### 4.2 рЃЏрЃарЃљрЃЋрЃљрЃџрЃ»рЃћрЃарЃљрЃЊрЃў рЃЌрЃљрЃЏрЃљрЃерЃў
- рЃњрЃљрЃўрЃЏрЃћрЃЮрЃарЃћрЃЌ рЃЌрЃљрЃЏрЃљрЃерЃў рЃарЃљрЃЏрЃЊрЃћрЃюрЃ»рЃћрЃарЃЏрЃћ
- рЃерЃћрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЌ рЃЦрЃБрЃџрЃћрЃЉрЃўрЃА рЃЎрЃЮрЃарЃћрЃЦрЃбрЃБрЃџрЃў рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃљ

### 5. рЃЊрЃўрЃърЃџрЃЮрЃЏрЃўрЃА рЃњрЃћрЃюрЃћрЃарЃљрЃфрЃўрЃўрЃА рЃбрЃћрЃАрЃбрЃўрЃарЃћрЃЉрЃљ

#### 5.1 рЃЊрЃўрЃърЃџрЃЮрЃЏрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ
- **рЃЏрЃЮрЃЦрЃЏрЃћрЃЊрЃћрЃЉрЃћрЃЉрЃў:**
  1. рЃЊрЃћрЃерЃЉрЃЮрЃарЃЊрЃўрЃЊрЃљрЃю рЃњрЃљрЃЊрЃљрЃЊрЃўрЃЌ рЃЊрЃўрЃърЃџрЃЮрЃЏрЃўрЃА рЃњрЃЋрЃћрЃарЃЊрЃќрЃћ
  2. рЃЊрЃљрЃљрЃГрЃўрЃарЃћрЃЌ "рЃЊрЃўрЃърЃџрЃЮрЃЏрЃўрЃА рЃњрЃћрЃюрЃћрЃарЃљрЃфрЃўрЃљ" рЃдрЃўрЃџрЃљрЃЎрЃА

- **рЃЏрЃЮрЃАрЃљрЃџрЃЮрЃЊрЃюрЃћрЃџрЃў рЃерЃћрЃЊрЃћрЃњрЃў:**
  - рЃЊрЃўрЃърЃџрЃЮрЃЏрЃўрЃА PDF рЃцрЃљрЃўрЃџрЃў рЃЕрЃљрЃЏрЃЮрЃўрЃбрЃЋрЃўрЃарЃЌрЃћрЃЉрЃљ
  - рЃЊрЃўрЃърЃџрЃЮрЃЏрЃў рЃерЃћрЃўрЃфрЃљрЃЋрЃА рЃАрЃгрЃЮрЃа рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃА (рЃАрЃљрЃ«рЃћрЃџрЃў, рЃњрЃЋрЃљрЃарЃў, рЃцрЃљрЃЎрЃБрЃџрЃбрЃћрЃбрЃў, рЃАрЃљрЃерЃБрЃљрЃџрЃЮ рЃЦрЃБрЃџрЃљ)
  - рЃЦрЃљрЃарЃЌрЃБрЃџрЃў рЃћрЃюрЃўрЃА рЃњрЃарЃљрЃЏрЃљрЃбрЃўрЃЎрЃљ рЃАрЃгрЃЮрЃарЃўрЃљ

#### 5.2 рЃЊрЃўрЃърЃџрЃЮрЃЏрЃўрЃА рЃЉрЃћрЃГрЃЊрЃЋрЃљ
- рЃњрЃљрЃ«рЃАрЃћрЃюрЃўрЃЌ рЃЕрЃљрЃЏрЃЮрЃбрЃЋрЃўрЃарЃЌрЃБрЃџрЃў PDF
- рЃерЃћрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЌ рЃцрЃЮрЃарЃЏрЃљрЃбрЃўрЃарЃћрЃЉрЃљ рЃЊрЃљ рЃерЃўрЃюрЃљрЃљрЃарЃАрЃў

## ­ЪћЇ рЃерЃћрЃфрЃЊрЃЮрЃЏрЃћрЃЉрЃўрЃА рЃбрЃћрЃАрЃбрЃўрЃарЃћрЃЉрЃљ

### API рЃбрЃћрЃАрЃбрЃўрЃарЃћрЃЉрЃљ
- **Health Check:** рЃњрЃљрЃЊрЃљрЃЊрЃўрЃЌ `/api/health` - рЃБрЃюрЃЊрЃљ рЃЊрЃљрЃљрЃЉрЃарЃБрЃюрЃЮрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЉрЃљрЃќрЃўрЃА рЃАрЃбрЃљрЃбрЃБрЃАрЃў
- **Faculties API:** рЃњрЃљрЃЊрЃљрЃЊрЃўрЃЌ `/api/faculties` - рЃБрЃюрЃЊрЃљ рЃЊрЃљрЃљрЃЉрЃарЃБрЃюрЃЮрЃА рЃцрЃљрЃЎрЃБрЃџрЃбрЃћрЃбрЃћрЃЉрЃўрЃА рЃАрЃўрЃљ

### рЃерЃћрЃАрЃљрЃФрЃџрЃЮ рЃерЃћрЃфрЃЊрЃЮрЃЏрЃћрЃЉрЃў
- **рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЉрЃљрЃќрЃўрЃА рЃЎрЃљрЃЋрЃерЃўрЃарЃў:** рЃерЃћрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЌ рЃџрЃЮрЃњрЃћрЃЉрЃў Vercel-рЃќрЃћ
- **Blob Storage:** рЃцрЃЮрЃбрЃЮрЃА рЃљрЃбрЃЋрЃўрЃарЃЌрЃЋрЃўрЃА рЃерЃћрЃфрЃЊрЃЮрЃЏрЃћрЃЉрЃў
- **рЃАрЃћрЃарЃЋрЃћрЃарЃўрЃА рЃерЃћрЃфрЃЊрЃЮрЃЏрЃћрЃЉрЃў:** 500-рЃўрЃљрЃюрЃў рЃАрЃбрЃљрЃбрЃБрЃА рЃЎрЃЮрЃЊрЃћрЃЉрЃў

## ­ЪЊі рЃбрЃћрЃАрЃбрЃўрЃарЃћрЃЉрЃўрЃА рЃЎрЃарЃўрЃбрЃћрЃарЃўрЃБрЃЏрЃћрЃЉрЃў

### рЃгрЃљрЃарЃЏрЃљрЃбрЃћрЃЉрЃўрЃА рЃЎрЃарЃўрЃбрЃћрЃарЃўрЃБрЃЏрЃћрЃЉрЃў
- РюЁ рЃарЃћрЃњрЃўрЃАрЃбрЃарЃљрЃфрЃўрЃљ рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА рЃцрЃЮрЃбрЃЮрЃА рЃљрЃбрЃЋрЃўрЃарЃЌрЃЋрЃўрЃЌ
- РюЁ рЃљрЃЋрЃбрЃЮрЃарЃўрЃќрЃљрЃфрЃўрЃљ рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА рЃњрЃћрЃюрЃћрЃарЃўрЃарЃћрЃЉрЃБрЃџрЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃЌ
- РюЁ рЃЊрЃћрЃерЃЉрЃЮрЃарЃЊрЃў рЃљрЃЕрЃЋрЃћрЃюрЃћрЃЉрЃА рЃАрЃгрЃЮрЃа рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃА
- РюЁ рЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃА рЃерЃћрЃерЃўрЃюрЃћрЃЉрЃљ рЃќрЃарЃЊрЃўрЃА рЃЦрЃБрЃџрЃћрЃЉрЃА
- РюЁ рЃЊрЃўрЃърЃџрЃЮрЃЏрЃў рЃњрЃћрЃюрЃћрЃарЃўрЃарЃЊрЃћрЃЉрЃљ рЃАрЃгрЃЮрЃарЃљрЃЊ
- РюЁ рЃДрЃЋрЃћрЃџрЃљ API рЃћрЃюрЃЊрЃърЃЮрЃўрЃюрЃбрЃў рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА

### рЃгрЃљрЃарЃБрЃЏрЃљрЃбрЃћрЃЉрЃџрЃЮрЃЉрЃўрЃА рЃЎрЃарЃўрЃбрЃћрЃарЃўрЃБрЃЏрЃћрЃЉрЃў
- РЮї рЃарЃћрЃњрЃўрЃАрЃбрЃарЃљрЃфрЃўрЃўрЃАрЃљрЃА рЃцрЃЮрЃбрЃЮ рЃљрЃа рЃўрЃбрЃЋрЃўрЃарЃЌрЃћрЃЉрЃљ
- РЮї рЃљрЃЋрЃбрЃЮрЃарЃўрЃќрЃљрЃфрЃўрЃљ рЃЋрЃћрЃа рЃ«рЃћрЃарЃ«рЃЊрЃћрЃЉрЃљ
- РЮї рЃЊрЃћрЃерЃЉрЃЮрЃарЃЊрЃў рЃљрЃа рЃљрЃЕрЃЋрЃћрЃюрЃћрЃЉрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃА
- РЮї рЃЌрЃљрЃЏрЃљрЃерЃў рЃљрЃа рЃќрЃарЃЊрЃўрЃА рЃЦрЃБрЃџрЃћрЃЉрЃА
- РЮї рЃЊрЃўрЃърЃџрЃЮрЃЏрЃў рЃљрЃа рЃњрЃћрЃюрЃћрЃарЃўрЃарЃЊрЃћрЃЉрЃљ рЃљрЃю рЃљрЃарЃљрЃАрЃгрЃЮрЃарЃўрЃљ

## ­Ъј» рЃерЃћрЃЏрЃљрЃ»рЃљрЃЏрЃћрЃЉрЃћрЃџрЃў рЃарЃћрЃЎрЃЮрЃЏрЃћрЃюрЃЊрЃљрЃфрЃўрЃћрЃЉрЃў

1. **рЃбрЃћрЃАрЃбрЃўрЃарЃћрЃЉрЃљ рЃЊрЃљрЃўрЃгрЃДрЃћрЃЌ** рЃарЃћрЃњрЃўрЃАрЃбрЃарЃљрЃфрЃўрЃўрЃЊрЃљрЃю рЃЊрЃљ рЃњрЃљрЃљрЃњрЃарЃФрЃћрЃџрЃћрЃЌ рЃЏрЃЌрЃћрЃџрЃў рЃцрЃџрЃЮрЃБрЃА рЃЏрЃўрЃДрЃЮрЃџрЃћрЃЉрЃўрЃЌ
2. **рЃДрЃЮрЃЋрЃћрЃџрЃў рЃАрЃфрЃћрЃюрЃљрЃарЃў** рЃњрЃљрЃўрЃЏрЃћрЃЮрЃарЃћрЃЌ рЃЏрЃўрЃюрЃўрЃЏрЃБрЃЏ 2-3-рЃ»рЃћрЃа
3. **рЃерЃћрЃфрЃЊрЃЮрЃЏрЃћрЃЉрЃўрЃА рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљрЃерЃў** рЃерЃћрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЌ рЃџрЃЮрЃњрЃћрЃЉрЃў рЃЊрЃљ API рЃћрЃюрЃЊрЃърЃЮрЃўрЃюрЃбрЃћрЃЉрЃў
4. **рЃгрЃљрЃарЃЏрЃљрЃбрЃћрЃЉрЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ** рЃерЃћрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЌ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЉрЃљрЃќрЃўрЃА рЃЏрЃЊрЃњрЃЮрЃЏрЃљрЃарЃћрЃЮрЃЉрЃљ
5. **рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃўрЃарЃћрЃЉрЃљ** рЃЕрЃљрЃўрЃгрЃћрЃарЃћрЃЌ рЃДрЃЋрЃћрЃџрЃљ рЃљрЃдрЃЏрЃЮрЃЕрЃћрЃюрЃўрЃџрЃў рЃърЃарЃЮрЃЉрЃџрЃћрЃЏрЃљ рЃЊрЃљ рЃњрЃљрЃЊрЃљрЃгрЃДрЃЋрЃћрЃбрЃўрЃџрЃћрЃЉрЃћрЃЉрЃў
